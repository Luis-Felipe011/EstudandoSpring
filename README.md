- Curso de Java e Spring Boot — Projetos e Fundamentos Essenciais
Este repositório concentra os projetos e atividades desenvolvidos ao longo da conclusão de um curso completo de Java e Spring Boot, voltado à consolidação de fundamentos sólidos de programação e à construção de aplicações modernas e bem estruturadas.
Durante o percurso formativo, foram explorados desde os princípios básicos da lógica de programação até o desenvolvimento de APIs RESTful robustas, utilizando os principais recursos do ecossistema Spring. O processo de aprendizagem foi fortemente orientado à prática, por meio da implementação de projetos reais, abrangendo aplicações desktop, sistemas web e integrações com bancos de dados.

- Principais competências desenvolvidas
Programação em Java desde o nível introdutório, com abordagem progressiva e estruturada
Aplicação dos conceitos de Programação Orientada a Objetos (POO) e Programação Funcional
Desenvolvimento de APIs REST utilizando Spring Boot
Integração com bancos de dados por meio de JDBC, Hibernate/JPA e SQL, com operações CRUD
Criação de interfaces gráficas utilizando JavaFX e estilização com CSS
Testes, consumo e validação de APIs com Postman
Implementação de mecanismos de segurança em aplicações web (CORS e CSRF)
Desenvolvimento e otimização de projetos com versões atuais do Spring Boot
Consumo de serviços externos e integração entre aplicações via Web Client
Utilização do Spring Tool Suite (STS) como ambiente de desenvolvimento
Construção de sistemas práticos, como uma aplicação para consulta de dados climáticos em tempo real

Este repositório evidencia a aplicação prática dos conhecimentos adquiridos e representa de forma consistente a evolução técnica alcançada ao longo do curso, refletindo o domínio progressivo das tecnologias e boas práticas adotadas no desenvolvimento de software com Java e Spring Boot.
